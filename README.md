# Mainline Website

Website: https://mainline.sh
Essay: https://mainline.sh/blog/why-coding-agents-need-repo-memory

- Markdown blog posts
- Static output for Vercel or Cloudflare Pages

## Development

```bash
npm install
npm run dev
```

```bash
npm run build
```

## Deploy

This repo is configured for Vercel static deployment:

```bash
vercel --prod
```
